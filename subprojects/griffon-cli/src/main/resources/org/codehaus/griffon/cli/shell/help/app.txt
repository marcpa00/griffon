DESCRIPTION
        app

	Runs the application in standalone mode.

SYNTAX
        app [options] [arguments] 

ARGUMENTS
        arguments
                Optional arguments to be passed to the application.

OPTIONS
        --debug
                Whether to run the application in debug more or not.
        --debug-addr
                Debugging address.
                (defaults to 127.0.0.1)
        --debug-port
                Debugging port connection.
                (defaults to 18290)
        --debug-suspend
                Whether to start the debug process in suspend mode or not.
                (defaults to n)
        --java-opts
                Additional JVM options.
        --jvm-opts
                Additional JVM options such as -javaagent.
        --non-interactive
                Controls if the shell can ask for input or not.
